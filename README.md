# 🍎 NutriCalc - Calculadora Nutricional

O **NutriCalc** é uma aplicação web desenvolvida para facilitar o cálculo de parâmetros nutricionais (como o IMC ou taxas metabólicas), oferecendo uma interface simples, rápida e intuitiva para o usuário.

> 🚀 **Veja o projeto no ar:** [Clique aqui para acessar o NutriCalc](https://diogo-abraga.github.io/Projeto-NutriCalc/)

---

## 📋 Sobre o Projeto

Este projeto foi criado com o objetivo de aplicar conceitos fundamentais de desenvolvimento web front-end, focando em manipulação de formulários, cálculos matemáticos via JavaScript e design responsivo.

### Funcionalidades:
- [x] Input de dados dinâmicos (Peso, Altura, Idade, etc).
- [x] Cálculo automático e preciso.
- [x] Interface adaptável para dispositivos móveis (Celular/Tablet).
- [x] Feedback visual imediato dos resultados


## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as "Três Torres" do Front-end:

* **HTML5:** Estruturação semântica de formulários e seções.
* **CSS3:** Estilização moderna, uso de Flexbox/Grid e efeitos de transição.
* **JavaScript (ES6+):** Lógica por trás dos cálculos, tratamento de eventos e manipulação do DOM.
